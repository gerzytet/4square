4square.py by gerzytet

tested on
Python 3.7.0 with Pygame 1.9.6 installed

HOW TO PLAY:
A pattern will flash on the 4 colored squares
After the flashing, remember the order they flashed in and repeat this order by clicking the squares
After each successful round, the pattern will get longer
if you click the wrong order, the game is over
The Score represents the amount of correct clicks total
